# 🚲 Sistema de Gestión de Bicitaxis CDMX

Proyecto desarrollado para la carrera de **Ingeniería en Computación (UNAM FES Aragón)**.  
Sistema Full-Stack para el control de conductores y mantenimiento de unidades.

## Tecnologías
* **Frontend:** HTML5, CSS3 (Figma Design), JavaScript (Vanilla).
* **Backend:** Node.js, Express.
* **BD:** MySQL 8.0.
* **Orquestación:** Docker & Docker Compose.

## Cómo ejecutar
1. Clona el repositorio.
2. Crea un archivo `.env` basado en los ejemplos vistos.
3. Ejecuta en la terminal:
   ```bash
   docker compose up --build
